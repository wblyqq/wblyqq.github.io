## 1、标题


```

```

## 2、分割线
```
- - -
* * *
一行只有符号，注意不要被识别成二级标题即可
```

## 3、斜体和粗体

```
*斜体*
**粗体**
***又斜又粗***
```

```

            + 四层

6、有序列表
===
```
列表嵌套只需在子列表中的选项前面添加两个或四个空格即可
1. 一层
    1. 二层
        1. 三层
    2. 二层
2. 一层
```

## 11、流程图
```

'name =>type: describe',其中type
主要有以下几种：
1、开始和结束：
2、输入和输出：input output
3、操作： operation
4、条件：condition
5、子程序： subroutine
